# Linked Data

Het idee van **Linked Data** is nieuwer dan dat van het **semantische web**, maar het is makkelijker om **Linked Data** te zien als een voortbouwing op de ideeën rond het semantische web. Linked Data is geen specificatie, maar een set van best practices die het makkelijker maakt om data te delen. ## Linked Open Data

De term **“Linked Open Data”** wordt ook steeds populairder. De groeiende hoeveelheid vrij beschikbare publieke data is een positieve evolutie, maar het is belangrijk om te onthouden dat Linked Data niet beperkt is tot open data.

Net zoals webservers en webpagina’s intern binnen een organisatie gebruikt kunnen worden om informatie te delen achter een bedrijfsfirewall, kunnen **Linked Data-technieken** ook intern grote meerwaarde bieden. Ze maken het makkelijker om data te delen tussen **interne systemen en datasilo’s**, zonder die data noodzakelijk publiek toegankelijk te maken.

Het semantische web en de concepten rond Linked data steunen op open standaarden en best practices om data en betekenis uitwisselbaar te maken. De fundamentele bouwsteen daarvan is **RDF**. In het volgende hoofdstuk zoomen we in op dit datamodel en bekijken we hoe feiten en relaties concreet worden vastgelegd.
